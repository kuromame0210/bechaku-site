# Requirements (Gate 0)

## Goal
- PC表示で左右の余白が大きすぎる問題を改善し、コンテンツをより大きく表示する。
- 40〜50代が閲覧するサイトとして適切か評価し、修正案を提示する。

## Scope
- `max-width` 指定箇所の棚卸し
- 余白・横幅の調整方針の策定
- UIの視認性/可読性の観点で評価と改善案を提示

## Constraints
- モバイルのレイアウトは原則維持
- 既存のデザインバランスが崩れない範囲で調整

## Open Questions
- 目標とする最大幅の目安（max-width: 1280pxで検討）
- 特に広げたいページ/セクションの有無
