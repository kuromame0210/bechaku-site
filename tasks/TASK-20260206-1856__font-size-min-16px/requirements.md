# Requirements (Gate 0)

## Goal
- PC表示時の文字サイズが小さく見える問題を改善し、全体の最低文字サイズを16px程度にする。

## Scope
- PCブレークポイントにおける本文・補助テキストの既定サイズの調整
- 既存のタイポグラフィ設計がある場合はそれに合わせて更新

## Constraints
- モバイルの文字サイズは原則維持（必要なら別途合意）
- 見出し階層のバランスは崩さない
- 既存デザインのトーンを維持

## Open Questions
- 「16px程度」は最小値の目安
- 対象ページ/セクションは全体
- 既存のタイポグラフィルールの有無
