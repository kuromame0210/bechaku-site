# 要件
- 依頼: お問い合わせフォーム画面を設計する。
- フォーム項目（必須）:
  - 会社名*
  - 担当者名*
  - 住所*（〒と郵便番号入力、郵便番号入力で住所自動入力の注記表示）
  - 電話番号*（例のプレースホルダーあり）
  - メールアドレス*（例: info@abcd.co.jp のプレースホルダー）
  - お問い合わせ種別*（選択肢: リバースエンジニアリング / 旧パーツや部品の復元 / オリジナル商品・ノベルティ部門 / その他）
  - 内容*
  - 「モデリングにあたっての注意事項に合意する」チェック*
- フォーム項目（任意）:
  - 添付ファイル（※3MBまで可能、最大3件）
- 添付ファイルの表示/操作:
  - 「選択されていません」の表示があること（3枠）。
- 画面に含める注記:
  - 「※郵便番号をご入力頂くと住所が自動入力されます。」
  - 「※3MBまで可能」
