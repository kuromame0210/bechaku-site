# 石垣さん向け要件サマリ

## 目的（ゴール）
- 3Dスキャナ/3Dプリンタの営業用サイトとして機能させる
- SEO経由の問い合わせを増やす
- URLを渡して「あとで見てください」が成立し、理解→相談に繋がる
- 営業資料として説明に使える

## 優先度
- 3Dスキャン/3Dプリントの営業が最優先
- 本業の会社情報はサブ（1ページで十分）

## 対象ページ
- トップページ
- 3Dスキャン
- 3Dプリント
- 会社案内

## ページごとの目的
- トップ: 何ができるかの全体理解と問い合わせ動線
- 3Dスキャン: 実物→データ化の理解と相談意欲の喚起
- 3Dプリント: データ→造形の理解と相談意欲の喚起
- 会社案内: 信頼担保の補足情報（本業の説明）

## 想定ターゲット
- 3Dスキャン/3Dプリントの依頼を具体的に検討している人
- 「こんな人におすすめ」を明確にし、該当者の問い合わせを増やす
- 目的に合わない人の問い合わせは減らしたい

## CTAの意図
- 問い合わせ（相談/依頼）に繋げる
- 見込みが高い人だけが連絡する状態を作る

## コンバージョン動線
- SEO/URL共有で流入 → 何ができるか理解 → こんな人におすすめで自分事化 → 問い合わせ

## 成果指標（方向性）
- 「あとで見てください」から相談に繋がる割合
- 問い合わせの質（目的が明確な人の比率）

## 必須で載せる情報（4項目）
- 実物 → データ化
- データ → 造形
- 旧パーツ復元
- 試作・量産
