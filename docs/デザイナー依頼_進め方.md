# デザイナー依頼の流れと方針

## 目的
- 今あるサイトを、デザイン的にもう一段上げたい
- 実装は自分でやるので、改善ポイントの整理と方向性の提案に力を貸してほしい

## 依頼の流れ
1) トップページのレビューを1万円で依頼（スマホ/PC両方）
2) レビュー提出 → 打ち合わせ → 継続判断
3) OKならトップの簡易作り直し（1〜2万円）を依頼
4) トップの出来を見て、残りページは自力で横展開 or 追加依頼
5) 追加依頼する場合はページ単価で合意して進行

## 依頼方針（要点）
- まず小額で相性確認、合わなければ止める
- 実装は自分が担当、納品は「作り直しの材料」重視
- 画像や参考例は具体例が分かる形で提示してもらう
- クライアントOKが最重要なので、方向性確認を挟む

## 依頼時の伝え方（短文）
- まずトップのレビューだけ1万円でお願いしたい
- レビュー内容を見て、継続するか決めたい
- 実装は自分でやるので、パーツ単位の提案でもOK
- 画像/絵は、具体例が分かる形で示してほしい
